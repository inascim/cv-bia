# cv-bia

CV toolkit for **Beatriz Almeida Giesteira** — six tailored resume versions in Portuguese and English, built in LaTeX and Markdown, targeting a logistics support role (Teclena Juncor, Leiria).

## Versions

| # | File | Language | Strategy |
|---|------|----------|----------|
| V1 | `v1-ats-otimizado-pt` | Portuguese | ATS-optimized — plain structure, high keyword density for online portals |
| V2 | `v2-moderno-profissional-pt` | Portuguese | Modern professional — polished layout for direct email/in-person delivery |
| V3 | `v3-transicao-carreira-pt` | Portuguese | Career transition — highlights transferable skills, addresses lack of direct logistics experience |
| V4 | `v4-ats-optimized-en` | English | ATS-optimized (English) — for international or multilingual recruiters |
| V5 | `v5-modern-professional-en` | English | Modern professional (English) |
| V6 | `v6-career-transition-en` | English | Career transition (English) — strongest narrative version |
| V7 | `v7-combinado-pt` / `v7-sem-fundo` / `v7-com-fundo` | Portuguese | Combined layout with and without photo background |

Each version exists as `.tex` (source), `.md` (content), and `.pdf` (output).

## Recommended strategy

- **Online job portals** (LinkedIn, Indeed, Net-Empregos): use **V1** (ATS)
- **Direct email to recruiter / in-person**: use **V3** (career transition, Portuguese)

See [`analysis-and-recommendations.md`](analysis-and-recommendations.md) for a full comparison, ranking, and cover letter strategy.
